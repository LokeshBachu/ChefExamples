How to install chef inside windows ?


https://downloads.chef.io/chefdk   

from this download the msi for windows and install it.

For the Chef inside ubuntu :

sudo apt-get update
sudo apt-get -y install curl

curl https://omnitruck.chef.io/install.sh | sudo bash -s -- -P chefdk -c stable -v 0.18.30

chef --version

create directory chef-repo

cd into chef-repo

vi test.rb

file '/tmp/motd' do
  content 'hello world'
end

chef-client --local-mode test.rb
